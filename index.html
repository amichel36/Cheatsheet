HTML/CSS Cheatsheet

HTML Elements

HTML, or HyperText Markup Language, is a language used for logical heirarchy.

Common HTML Tags

Tag Usage
HTML <html> This tag is used to identify where the html code starts and ends.
Head <head> This tag defines the head section of the HTML. This section initializes certain things in the page.
Body <body> This tag defines the body section. This section includes the structure of the page.
Header <h1-h6> This tag is used for different sized headers in a document.
Div <div> This tag is used dividing sections in a document.
Paragraph <p> This tag creates a paragraph section for general text in a page.

Tables

Tag Usage
Table <table> This tag creates a table structure in html.
Table Head <thead> This tag creates a head in a table structure.
Table Body <tbody> This tag creates a body in a table structure.
Table Header <th> This tag designates a header in a table element.
Table Row <tr> This tag is to add rows to a table.
Table Data <td> This tag adds data to a table cell.
